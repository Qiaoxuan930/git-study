1.学习资料来源于腾讯云开发者社区，相关链接：https://cloud.tencent.com/developer/article/2070991
2.实践流程：在百度上搜索Git Bash并安装，在D盘创建Git-study文件夹，打开Git Bash，切换到该文件夹，执行git init命令，即可初始化一个Git仓库。创建GitHub账号，在Git Bash中执行git config --global user.name "你的姓名"和git config --global user.email "你的邮箱"，即可配置Git用户名和邮箱,在Github上创建一个仓库，然后执行git remote add origin +https://github.com/Qiaoxuan930/git-study 关联本地仓库和远程仓库。
3.提交内容
  1. 第一次提交：创建项目，实现基础菜单和添加功能
  2. 第二次提交：新增平均分计算和最值查找功能
  3. 第三次提交：优化交互提示，完善异常处理
4.遇到的问题：执行git config --global user.name "你的姓名"和git config --global user.email "你的邮箱"命令时，无法配置成功，在.git文件夹的config文件中查看，发现user.name和user.email没有配置，需要手动添加。
5.学习心得
通过本次学习，我了解了Git的基本概念和操作，以及如何关联本地仓库和远程仓库。我学会了如何使用Git进行版本控制，以及如何在团队中合作开发。
